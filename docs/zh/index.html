關於我們

成立於2009年，台灣色彩與影像科技 (TCIT) 在影像、色彩與智能分析領域深耕數十年，同時提供創新科技結合新生活形態的應用，致力於滿足人類對於安全、便利跟快樂的需求。

我們以軟體研發、系統整合、網路應用服務為發展主軸，擅長影像分析技術、智慧辨識與創新服務，實際應用包含了人臉辨識與監控的系統整合、客層分析、數位看板廣告的觀眾效益量測、區域偵測以及即時門鎖監控系統！ TCIT 同時具備了核心技術與專業整合能力，為合作廠商提供跨平台且全方位的解決方案，共同邁向嶄新的物聯網及 AI 智慧時代！

協助企業將AI與軟硬體結合並提升產品價值

-演算法函式庫開發 (Algorithm Library for specify platform)
